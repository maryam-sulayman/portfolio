Weather Project

In this project, I was asked to build a weather app. This was a challenging project but I thoroughly enjoyed every aspect of it. During this 8-week workshop, I learned advanced CSS, HTML & Java-script. I equally learned about API integrations, the bootstrap framework & various external libraries used in the world of coding. At the end of this workshop, I was able to create a functional weather app which gives the users location, temperature, date, humidity & wind-speed. This app also has a search input where the user can search for any city anywhere in the world.

Clock Project

For this project, I was asked to build an app displaying the current time across various cities. During this workshop, I learned about HTML inputs and was taught to use moment.tz() to find the current time in different timezones. This workshop was great and I was able to create a clock app which displays the current time for a list of some cities. The user is also able to find out the time in their current timezone by clicking on the location icon next to the select menu.
