A look at my work...

Pasta Project

I entered the world of front-end development by understanding how HTML, CSS and JavaScript work. In this project, I built a webpage about my favourite type of food with some added features.

Weather Project

For this project, I built a Vanilla Javascript weather application. This was a challenging one but I thoroughly enjoyed every aspect of it. I became skilled using API's, Bootstrap, real-life development workflow & hosting on Github. At the end of this workshop, I built a fully-functional weather app which also gives the users current weather information in their area.

React Weather Project

I upgraded my existing weather application by re-building it in REACT. This was fun and exciting, I was able to organise my project better by creating components to handle different aspects of the application.

Home-town Project

I delved into the realm of Responsive Web Design by learning the art of optimizing websites for flawless display using the mobile-first strategy. In the context of this particular project, I built a website dedicated to my hometown, Lagos. I integrated the use of grids, columns, and embeds to build an aesthetically pleasing layout, whilst maintaining responsiveness to improve user interaction.

Clock Project

I built an app displaying the current time across various cities in this project. Users have the ability to access the time in their immediate time zone by interacting with the location icon adjacent to the selection menu. This application was built utilizing the JavaScript library from the Moment timezone.

Dictionary Project

In this project, I built a dictionary application in REACT. This application allows users to look up words, subsequently retrieving comprehensive information such as definitions, phonetics, synonyms, and corresponding images associated with the word.
